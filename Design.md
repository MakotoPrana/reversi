# ゲームデザイン
## 基本ルール
・8✕8の盤上に２人で交互に白と黒の石を置く。
・石は挟まれると相手に取られる。

## ゲームの終了条件
・マスがすべて埋まる。
・石を置けるところがなくなる。

## ゲームの勝敗
・板状の石の数が多い方が勝ち。

# UI設計
## インターフェース
CLIインターフェース上から実行可能

## コマンド
### 新しくゲームを始める
php reversi start

ゲームを開始する。黒が先行でゲームを開始する。
以降、白と黒で１ターンずつ交代で操作する。

### 1ターンの流れ
・任意の位置に石を置く
 ・置けない位置を指定した場合はエラー
 ・置いた石の状態に応じて石を裏返す。
 ・終了条件を判定する。
